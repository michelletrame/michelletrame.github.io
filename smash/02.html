Project

Over the course of this class, you are going to build a personal website that you can use to tell the world about yourself and whatever you are passionate about.

  1. Take a moment to think of a couple of ideas. They can be funny or serious or anything else.
  2. Close your eyes and envision what type of person might visit your website in as much detail as possible. How will they want to interact with your site? How will you present yourself to these visitors?
  3. Ask some of the other scholars about their ideas. Also take some time to get to know them a little.

Exercise: About Me

Let's write an "About Me" page introducing yourself.

  1. Correctly set up the HTML document with a head and body tag
  2. The page should have a title tag in the head
  3. Use a few "horizontal rule" delimiters or line break tags as appropriate
  4. Use at least three different sizes of header tags
  5. Use ordered and unordered lists
  6. Use some <em> and <b> tags
  7. Use some <br> and <hr> tags
  8. Include profile photos
  9. In fact, we'll need plenty of images (on your computer or on the web). Users love images!
  10. Can you figure out how to resize the images?" Use google!
  11. Link to some external websites
  12. Create a "navigation bar" to allow users to click between the landing page and the "about us" section. See below for more info on the nav structure.
  13. We need easter eggs! Have at least a few SECRET COMMENTS that visitors can only see if they view your source code!

Some things to note:

This link tag has an attribute whose name is href and whose value is a url.

Attributes provide additional information about an element. They appear on the opening tag of the element and are made up of two parts: a name and a value, separated by an equals sign.

<img src="https://i.imgur.com/81qyN1y.jpg" alt="This text will show up, only if the image doesn't (also good for screen readers)">

Image tags have an attribute named src whose value is the location of the image to be displayed.
